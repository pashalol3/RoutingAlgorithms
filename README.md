# Зависимости:
- SkiaSharp (рисует картинки)

# Для запуска: 
1. Установите SDK.
2. 
   ```cmd
   cd ../RoutingAlgorithms	
   dotnet restore
   dotnet run


```
Program.cs
var nodes = GraphGenerator.RandomTree(numNodes:6, connectionProbability: 0.3f);
var source = nodes.First();
var dest = nodes.Last();
var result = SAMCRA.FindRoute(  source: source, 
                                destination: dest, 
                                routeMaxCost:2f, 
                                routeMaxTime:2f
                             );
```
